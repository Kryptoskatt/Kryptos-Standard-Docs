# Live Demo Apps

Explore interactive demos built with Kryptos Connect. Each demo connects a real wallet and showcases a specific API — no setup required.

**Demo hub:** [demo-connect.kryptos.io](https://demo-connect.kryptos.io/)

---

## Transactions

Unify every transaction into one clear, consistent view.

> Kryptos transforms messy, diverse transactions into standardized, easy-to-understand data.

Connect your wallet to see all on-chain and exchange transactions normalized into a single, consistent format — with type classification, fee breakdown, and fiat values.

**API used:** [Transactions API](/docs/api/transactions)

[Try the Transactions Demo →](https://demo-connect.kryptos.io/demo/transactions)

## NFT Portfolio

Track every NFT across wallets and marketplaces.

> Track NFT holdings across wallets and marketplaces with a single normalized API.

View all NFTs held across multiple wallets and chains in one place, with metadata, collection info, and marketplace pricing pulled via a single API call.

**API used:** [NFT Holdings API](/docs/api/nft-holdings)

[Try the NFT Portfolio Demo →](https://demo-connect.kryptos.io/demo/nfts)

## Holdings

Aggregate all token balances into one normalized view.

> Kryptos aggregates token balances from every source into a normalized holdings API with prices, ROI, and per-wallet breakdowns.

See token balances aggregated across wallets and exchanges, enriched with live prices, cost basis, ROI, and unrealized gains per asset.

**API used:** [Holdings API](/docs/api/holdings)

[Try the Holdings Demo →](https://demo-connect.kryptos.io/demo/holdings)

## DeFi

All DeFi positions in one unified dashboard.

> Kryptos brings all your DeFi investments into one unified dashboard — tracking staking, lending, and yield farming across multiple protocols with real-time valuations.

Connect your wallet to see all DeFi positions (staking, lending pools, yield farms) normalized across protocols with real-time USD valuations.

## Portfolio Insights
